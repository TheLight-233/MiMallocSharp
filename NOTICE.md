# Attribution and project status

MiMallocSharp is an independent community project and a C# translation/port of
[mimalloc](https://github.com/microsoft/mimalloc), the original allocator from
Microsoft Research and its contributors.

This repository is not the official mimalloc repository and is not affiliated
with or endorsed by Microsoft. Please consult the upstream project for the
canonical C implementation, documentation, release history, and license
notices:

- Upstream repository: https://github.com/microsoft/mimalloc
- Upstream documentation: https://microsoft.github.io/mimalloc/

The translated code in this repository is provided under the MIT License. The
upstream project and its original copyright notices remain the authoritative
source for the original mimalloc implementation.
